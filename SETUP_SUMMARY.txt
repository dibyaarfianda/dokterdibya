ğŸ“ How to Connect to the Project
================================

FOR FIRST-TIME SETUP:
--------------------
Read: QUICKSTART.md (if you want to get running quickly)
   or CONNECTION_GUIDE.md (for complete instructions)

ESSENTIAL STEPS:
1. Install Node.js v16+ and MySQL v8.0+
2. Clone: git clone https://github.com/dibyaarfianda/dokterdibya.git
3. Install: cd staff/backend && npm install
4. Database: mysql -u root -p -e "CREATE DATABASE dibyaklinik"
5. Config: cp staff/backend/.env.example staff/backend/.env
          (Edit .env with your database credentials)
6. Start: node staff/backend/server.js
7. Access: http://localhost:3000/staff/public/index-adminlte.html
